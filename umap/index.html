Standard UMAP — 1-Skeleton
The standard UMAP algorithm explicitly works with the fuzzy 1-skeleton only — vertices (0-simplices) and weighted edges (1-simplices). The higher-order simplices exist in the theory but are discarded: the algorithm reduces the fuzzy simplicial set to a weighted graph. This makes it computationally tractable but means only $H_0$ (cluster connectivity) is directly encoded. Loops and voids in the data must be inferred indirectly.
Thesis Extension — 2-Skeleton
The thesis extends the construction to include 2-simplices (filled triangles), whose fuzzy membership is set by the minimum edge weight of their three bounding 1-simplices — the simplicial condition $\mu(\sigma) \leq \mu(\partial\sigma)$. This directly encodes $H_1$ (loops and cycles), giving a richer topological representation. The cross-entropy objective is then optimized over the full 2-skeleton, not just the graph.

Mathematical Framework

Topology · Manifold Learning
01
The Manifold Hypothesis

Real high-dimensional data — images, genomic sequences, financial returns — does not fill its ambient space uniformly. It tends to concentrate near a low-dimensional manifold embedded in the high-dimensional space. A 64×64 image lives in ℝ4096, but the set of natural images forms a manifold of much smaller intrinsic dimension.

Dimensionality reduction exploits this: if the true intrinsic dimension is d ≪ D, we should be able to represent the data faithfully in ℝd. The challenge is doing so while preserving the geometric and topological structure of the manifold.

02
Riemannian Geometry — Local Metric Estimation

UMAP begins by assuming the data lies on a Riemannian manifold with an unknown metric. It estimates the local metric structure from the k-nearest neighbors of each point. For each point xi, the metric is normalized so that the distance to the nearest neighbor is exactly 1:

$$d_i(x_i, x_j) \;=\; \|x_i - x_j\| - \rho_i$$

where ρi is the distance to the nearest neighbor. This normalization makes the metric locally constant — every point "sees" its neighborhood at the same scale — which is essential for handling datasets with varying densities.

03
Fuzzy Simplicial Sets — The Topological Representation

The local metrics are converted into a fuzzy simplicial set — a generalization of a simplicial complex (think: a graph, triangles, tetrahedra...) where membership has a probabilistic weight between 0 and 1. The weight of the edge between points xi and xj is:

$$w(x_i,\, x_j) \;=\; \exp\!\left(\frac{-(d_i(x_i, x_j) - \rho_i)}{\sigma_i}\right)$$

where σi is chosen so the sum of weights equals log2(k) — enforcing a consistent notion of "k neighbors." Multiple local views (one per point) are combined into a single global fuzzy set via a symmetric union:

$$w_{\mathrm{sym}}(i,j) \;=\; w(i,j) + w(j,i) - w(i,j)\cdot w(j,i)$$

This is the fuzzy set analogue of taking the union: P(A ∪ B) = P(A) + P(B) - P(A)P(B) for independent events.

04
Low-Dimensional Embedding via Cross-Entropy Optimization

UMAP finds a low-dimensional layout Y = {y1, ..., yn} ⊂ ℝd that best matches the high-dimensional fuzzy structure. The low-dimensional edge weights are modeled using a smooth family of curves:

$$v(y_i,\, y_j) \;=\; \left(1 + a\,\|y_i - y_j\|^{2b}\right)^{-1}$$

Parameters a and b are fit to approximate a smooth step function. The objective is to minimize the cross-entropy between the two fuzzy sets — the high-dimensional w and the low-dimensional v:

$$\mathcal{C} \;=\; \sum_{i,j}\Bigl[w\log\frac{w}{v} + (1-w)\log\frac{1-w}{1-v}\Bigr]$$

This is minimized with stochastic gradient descent using negative sampling for efficiency. Intuitively: attract connected points, repel disconnected ones. Unlike t-SNE's KL divergence, the cross-entropy treats attraction and repulsion symmetrically, which is why UMAP better preserves global structure.

05
Connection to the Thesis — UMAP via Category Theory

The thesis frames UMAP rigorously using category theory. The central object is the fuzzy singular complex functor $\widetilde{\mathrm{Sing}}_X$, which takes a finite metric space $(X, d)$ and produces a fuzzy simplicial set encoding all pairwise distance relationships. This functor has a left adjoint — the metric realization functor — giving an adjunction between fuzzy simplicial sets and extended pseudo-metric spaces.

The key insight is that the fuzzy simplicial set construction approximates the Čech complex of the underlying manifold. By the Nerve theorem, this captures the manifold's homology — its connected components ($H_0$), loops ($H_1$), and voids ($H_2$). UMAP's embeddings therefore preserve persistent homology in a formal sense: clusters, holes, and voids in the plot correspond to genuine topological features of the data.

The thesis also frames the local connectivity condition — setting $\rho_i = \min_j d(x_i, x_{ij})$ so the nearest neighbor is at distance zero — as a requirement that each $0$-simplex is the face of some $1$-simplex with fuzzy membership strength $1$. This ensures the resulting topological structure is locally connected, a categorical condition rather than a geometric one.

R1
The Sector Rotation Loop — A Real $H_1$ Generator in Market Data

S&P 500 sectors exhibit a well-known rotation cycle in their return correlations: Technology → Consumer Discretionary → Financials → Utilities → Technology. In the correlation metric space, this cycle forms a genuine $H_1$ generator — a loop that cannot be contracted to a point. The standard UMAP 1-skeleton, which only builds a weighted graph, cannot distinguish this loop from a dense cluster or an artifact of noise. A graph admits cycles of edges, but without 2-simplices, there is no notion of "filled interior" to compare against.

The 2-skeleton resolves this: these sectors form a ring of triangles whose interior remains hollow — the simplicial condition $\mu([i,j,k]) = \min(w_{ij}, w_{jk}, w_{ik})$ assigns low membership to any triangle spanning the gap, leaving the loop boundary as a non-trivial 1-cycle. This is precisely the $H_1$ generator the 1-skeleton cannot certify. Hover over sectors in the 2-skeleton panel to inspect triangle memberships.

R2
Why This Matters for Quantitative Finance

Sector rotation is a core concept in macro investing: capital flows from growth sectors (Tech, Consumer Discretionary) to defensive sectors (Utilities, Consumer Staples) as the business cycle shifts. Detecting the loop topology of this rotation has direct implications — a $H_1$ generator with long persistence in a barcode means the rotation cycle is structurally stable, not just a transient correlation spike. The 2-skeleton, by certifying the loop algebraically rather than just observing correlated edges, provides a stronger invariant for strategy design and risk decomposition.

Concretely: a risk model built on the 1-skeleton might merge growth and defensive sectors into a single cluster during a high-correlation period, missing the rotation loop entirely. The 2-skeleton preserves the loop structure because the hollow interior persists as long as the minimum triangle weight stays low — a geometric fact encoded by the simplicial condition, not a modeling choice.
